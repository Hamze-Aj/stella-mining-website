# Image Requirements

This document lists all the images needed for the Stella Mining website.

## Required Images

### Main Images
- `hero-mining.jpg` - Hero section background (recommended: 1920x1080px or larger)
- `about-company.jpg` - About section image (recommended: 800x600px)
- `company-profile.jpg` - Company profile section (recommended: 800x600px)

### Project Images
- `project-dembela.jpg` - Dembela project (recommended: 600x400px)
- `project-sali.jpg` - Sali project (recommended: 600x400px)
- `project-namamota.jpg` - Namamota project (recommended: 600x400px)

### Team Images
- `team-member-1.jpg` - CEO photo (recommended: 400x400px, square)
- `team-member-2.jpg` - COO photo (recommended: 400x400px, square)
- `team-member-3.jpg` - CFO photo (recommended: 400x400px, square)

### Gallery Images
Place in `gallery/` directory:
- `mining-operation-1.jpg` - Mining operation photo
- `mining-operation-2.jpg` - Mining equipment photo
- `mining-operation-3.jpg` - Site exploration photo
- `team-work-1.jpg` - Team collaboration photo
- `team-work-2.jpg` - Field operations photo
- `environment-1.jpg` - Environmental protection photo
- `environment-2.jpg` - Site rehabilitation photo
- `community-1.jpg` - Community engagement photo
- `community-2.jpg` - Local community partnership photo

## Image Guidelines

- **Format**: JPG or PNG
- **Optimization**: Compress images before uploading for better performance
- **Aspect Ratios**: Maintain consistent aspect ratios for similar image types
- **Alt Text**: All images should have descriptive alt text (already included in code)
- **File Size**: Keep images under 500KB when possible for web performance

## Placeholder Images

If you don't have images yet, you can:
1. Use placeholder services like placeholder.com or unsplash.com
2. Use stock photos from sites like Unsplash or Pexels
3. Create simple colored placeholder images

Example placeholder URL format:
- `https://via.placeholder.com/1920x1080/1f2937/ffffff?text=Stella+Mining`
